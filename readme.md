# LockInLauncher
 
**One command. Zero distractions. Full focus.**
 
LockInLauncher is a CLI-based productivity automation tool that eliminates distractions and instantly configures a focused workspace, helping users stay locked into deep work.
 
---
 
##  Features
 
-  **App/Site Blocking** — Blocks distracting websites and apps
-  **App Launcher** — Automatically opens the apps you need for your session
-  **Focus Timer** — Built-in focus timer to keep you on track
